# Cart Promotion Displayed

### 

## Javascript Code
```js
window.dataLayer = window.dataLayer || [];
dataLayer.push({ ecommerce: null });  // Clear the previous ecommerce object.
dataLayer.push({
  "event": "view_promotion",
  "detailed_event": "Cart Promotion Displayed",
    "ecommerce": {
        "items": [
            {
                "item_id": "<item_id>",
                "item_name": "<item_name>"
            }
        ]
    }
});
```
